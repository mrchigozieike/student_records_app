# Project Overview

This project is a **Student Database API** built using **Python, Flask, and SQLite**.  
The application demonstrates how software can interact with a **relational database using SQL** to perform CRUD operations.

The API allows users to:

- Add students
- View students
- Update student records
- Delete students
- View aggregate statistics about students

The API also includes **interactive documentation using Swagger (OpenAPI)**, allowing users to test endpoints directly from the browser.
